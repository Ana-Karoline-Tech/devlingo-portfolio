## 📝 Sobre o Projeto
O DevLingo é uma plataforma de ensino gamificada de alta fidelidade, inspirada no Duolingo, criada para transformar o aprendizado de tecnologia em uma experiência imersiva. Este projeto demonstra a aplicação de padrões avançados de desenvolvimento Frontend e integração robusta com serviços de Backend-as-a-Service.

## 🚀 Tecnologias Utilizadas
*   **React 19 + TypeScript:** Interface com tipagem estrita para máxima segurança.
*   **TanStack Router:** Gerenciamento de rotas moderno e 100% type-safe.
*   **Supabase:** Autenticação e Banco de Dados Relacional (PostgreSQL).
*   **Tailwind CSS:** Estilização responsiva e Pixel Perfect.
*   **Lucide React:** Ícones consistentes e elegantes.
*   **Zod:** Validação rigorosa de esquemas e dados.

## ✨ Funcionalidades em Destaque
*   🎯 **Gamificação Completa:** Sistema de XP, Níveis e Vidas para engajamento.
*   ⚡ **Sincronização em Tempo Real:** Atualização via WebSockets (Supabase Realtime).
*   🛤️ **Trilha de Aprendizado:** Lições progressivas desbloqueadas por desempenho.
*   📊 **Feedback Inteligente:** Telas de sucesso e recuperação baseadas na precisão do aluno.
*   🔐 **Autenticação Segura:** Fluxo de Sign-in/Sign-up integrado ao Supabase Auth.

## 🛠️ Arquitetura e Engenharia
*   **BaaS (Supabase):** Gestão estratégica de servidor com robustez PostgreSQL.
*   **RPC (Remote Procedure Calls):** Funções PL/pgSQL executadas no banco para garantir integridade atômica na atualização de XP.
*   **Roteamento Type-Safe:** Eliminação de erros de navegação em tempo de compilação.
*   **Gerenciamento de Estado:** Uso de Context API e Hooks customizados.
